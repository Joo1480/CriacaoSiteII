Comandos GitHub

  • Git Fetch --all

    • Sincroniza os arquivos locais e do github

  • Git Pull

    • Dowload das atualizações dos servidores remotos

Finalização das Atividades:

  • Git Status

    • lista as modificações

  • Git Add .

    • Sobe todos os arquivos para a nova versão

  • Git Commit -m "Texto"

    • Monta as modificações para envio ao servidor remoto

  • Git Push

    • Faz o upload das modificações do servidor local para o servidor remoto
